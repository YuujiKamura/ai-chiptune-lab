# BGM 試作解析レポート v2：Broken Beat

`out/bgm_v2.wav` ("Broken Beat") を解析し、v0 との比較評価を行いました。

### 1. リズムの変則性と正当性
- **評価：伝わっているが「破綻」の境界線上。**
- **音響的特徴：** v0 の均等なパルスから、鋭く不規則なピークの連続に変化。
- **印象：** シンコペーションが強烈すぎて、1拍目の重心（Downbeat）が見失われやすく、聴き手に高いリズム解釈能力を要求する「グリッチ」に近い構成。

### 2. チャンネル間の Groove
- **評価：会話は成立しているが、BASS が「浮いている」。**
- **BASSの問題：** 指摘通り Bass の位置が極めて不安定。Kick（Drums）と Bass の発音タイミングが一致しない箇所が多く、低域の核が分散している。
- **対話感：** Lead と Harm の裏拍ステイブは噛み合っているが、Bass が独自のシンコペーション（6, 3, 5, 2等）に走りすぎ、アンサンブルとしてのまとまり（Tightness）に欠ける。

### 3. 格闘ゲームとしての適性
- **タイトル向き：不向き。** 操作開始前の「待ち」としては落ち着きがない。
- **ステージ向き：非常に面白い。** トリッキーなキャラクターの専用ステージや、サイバーパンク的・実験的な演出には v0 より適している。
- **総評：** 正統派格ゲーから「インディー・尖った作品」へとジャンルがシフト。

### 4. 具体的な修正案（v3 への道）
- **案1（BASSの重心固定）：** 奇数小節の 1拍目（Step 1）と 3拍目（Step 9）は Bass を Kick と同期させ、楽曲の重心を安定させる。
- **案2（Leadの存在感回復）：** Duty 0.125 で細くなった Lead が、強まった Bass に埋もれがち。Duty 0.25 に戻すか、音量を 1.5dB 上げてメロディを明瞭にする。
- **案3（ドラムの強弱）：** すべての Noise ノートが同音量のため「点の暴力」に聴こえる。弱打（Ghost Note）を混ぜてリズムに「うねり」を作る。

### 5. 致命的な欠陥
- **低域の重心喪失：** 楽曲の Root を支えるべき Bass が 1拍目にいないことが多く、音楽的な重心が浮き上がってしまい、聴き手に不安感を煽る構成になっている。
